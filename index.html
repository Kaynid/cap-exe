STREETWEAR ARCHIVE // EST. 2003
// ORIGIN FILE

It started in a basement in 2003. No label. No backing. Just a kid with a fitted cap, a bag of pearls, and a needle.

He didn't call it fashion. He called it a file. Every cap was a program — hand-built, one of one, released into the streets like software into the internet.

// THE NAME

CAP.EXE was named after the way he thought about his work. Not a product. An executable. You wear it — it runs. It changes how people see you. You become the program.

The early internet had this energy — anything was possible if you knew where to look. The underground forums. The limited drops. The culture that lived between the pixels.

That's what CAP.EXE is. A digital archive of a physical era that most people forgot.

// THE PROTOCOL

Every cap is built by hand. Every piece ships with a patch number — 1/25, 1/10, 1/8 — because nothing here is mass produced. When it's gone, it's gone. The file closes.

Pearl studs. Lace. Rosary chains. Gold jewels. These aren't decorations. They're commits. Each one is a decision that can't be undone.

// THE MISSION

Not a hat. A file.
Not a brand. An archive.
Not streetwear. A protocol.

CAP.EXE exists to prove that the most powerful thing you can wear is something that could only have been made once.
